    sublrc-converter

SUBLRC Converter adalah sebuah program yang dirancang untuk mengkonversi format teks pada file lirik (LRC) ke format teks pada file subtitle (ASS) ataupun sebaliknya. Selain itu juga terdapat fitur untuk menyesuaikan waktu mulai (start time) ataupun waktu berakhir (end time) pada file subtitle ASS untuk digunakan pada aplikasi aegisub. Alat ini membutuhkan aplikasi Microsoft Excel sebagai mesin pemroses utama.
